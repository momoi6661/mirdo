# 《废土微光：铁皮与小空》 (Wasteland Glimmer) - 游戏设计与开发文档

## 一、 核心设定 (Core Concept)
- **主角 (玩家)**：战前遗留的护卫机器人（代号 R-7）。负责避难所的日常维护与物资分配。
- **NPC (AI伴侣)**：人类幸存少女“小空”。拥有独立的生理属性（饱食度、水分）与情感维度（心情值、好感度）。
- **游戏目标**：在地下避难所中，通过合理利用有限物资，与小空进行对话交互，提升好感度并确保其在 15 天内生存。

## 二、 核心系统集成

### 1. AI 智能交互目标 (核心研究内容)
- **对话处理**：调用后端大模型实现非线性、高自由度的自然语言对话。
- **动作驱动**：AI 根据语义自动触发 NPC 的 3D 动作（如点头、挥手、走向特定位置）。
- **环境感知**：AI 能通过“环境快照”识别玩家背包、储物箱内的物资状态，实现“见物叙事”。
- **情感养成**：对话内容会动态改变好感度数值，进而影响 NPC 的交互态度和后续剧情。

### 2. 已实现基础系统
- **背包系统**：支持物资拾取、查看与移交。
- **箱子系统**：实现物资的存储与持久化管理。
- **存档系统**：保存当前天数、时间、NPC 状态及交互记忆。
- **交互框架**：基于摄像机射线的 3D 物体检测与 UI 响应。

## 四、 后端 API 交互规范

**请求 (Godot -> Server):**
```json
{
  "day": 1,
  "npc_stats": {"hunger": 50, "mood": 60, "favor": 10},
  "context": {
	  "box": ["bread", "water"], 
	  "backpack": ["photo_album"]
  },
  "player_input": "小空，你看我找到了什么？"
}
```

**响应 (Server -> Godot):**
```json
{
  "text": "这是相册吗？在这个避难所里竟然还能看到这种东西...",
  "emotion": "surprised",
  "action": "walk_to_player",
  "stat_change": {"mood": 15, "favor": 5}
}
```
