# Mirdo Short Motion Prompts

> Character: Mirdo  
> Style: short Kimodo-style prompts.  
> Rule: prompts start with `A person...`; keep them simple.

## Notes

- For loop actions, use prompts containing `seamless loop`.
- For movement loops, root motion is allowed.
- Keep prompts short; add details only if a specific generation fails.
