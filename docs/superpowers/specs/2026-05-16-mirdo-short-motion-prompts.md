# Mirdo Short Motion Prompts

> Character: Mirdo  
> Style: short Kimodo-style prompts.  
> Rule: prompts start with `A person...`; keep prompts short.  
> Each prompt includes a Chinese meaning line for quick reading.  
> Target: compact but expressive core set.
