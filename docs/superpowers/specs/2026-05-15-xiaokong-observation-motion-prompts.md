# 小空观察型动作提示词（允许位移 / Root Motion 友好）

> 说明：
> - 动作名使用英文
> - 提示词使用中文
> - 移动类循环动作允许自然位移和根运动，适合后续用 Root Motion 归位
> - 循环动作统一强调首尾节奏连续，而不是强制原地

## 通用前缀

```text
二次元少女体型，穿短袖校服连衣裙，动作自然克制，适合裙装，站姿和步态稳定。手部动作以观察、确认、接近、轻触目标为主，幅度小而明确。
```
